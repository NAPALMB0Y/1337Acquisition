1337Inquisition is a Windows based forensics tool which utilizes basic Windows commands 
and various Sysinternals tools to pull information from a system. It also contains the tools EDD
and DumpIt to check for encrypted drives and to dump the RAM. This tool is designed to be run 
as a standard user, if you want to run the tool with admin privileges, you must change the .bat file to 
have the full file paths when it calls each tool.
